2. 思考アプローチの違い

時間軸の差：DeepSeek事例

私の視点：「将来のリスク」

  • 「DeepSeekがDDoSくらったニュース」
  • 「社員が将来アプリを使う可能性」
  • 「利用規約の潜在的リスク」
  • → 「注意喚起したほうがいい」

上司の視点：「現在の事実」

  • 「DDoSじゃなく利用者増加によるサーバダウン」
  • 「今使ってる人はいない」
  • 「規約が危ないエビデンスは？」
  • → 現状の事実確認を優先

お願い：時間軸・具体/抽象の違いを意識した協業

私の「将来予測」を「エビデンスのない無価値な話」ではなく、
「未来軸の抽象的な目標設定のための話」として捉えていただきたい

AIプロジェクトでも、将来の可能性やリスクを考慮することで、コストカット以上の成果が期待できます

3. 最近のAI観

新しい可能性への視点

現時点でのAI観

  • AIはコストカットだけでなく、新しい可能性を広げるもの
  • AIプロジェクトを通じてこの価値観を示したい

従来のAI観

  • コスト削減
  • 業務効率化
  • 自動化
  • 人員削減

目指すAI観

  • コスト削減 + 新しい価値創造
  • 効率化 + ユーザー体験向上
  • 自動化 + 人間の創造性拡張
  • 人員削減 → 人材の価値向上

詳細はレクチャー会後に

  • 具体的なデータや事例を揃えて相談したい
  • AIの可能性とリスクを包括的に把握した上で議論を深めたい
